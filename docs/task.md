# ТЗ: лендинг (сертификация / квиз-воронка)

## Ключевые принципы

- Акцент на CTA "Рассчитать"
- Квиз визуально главный блок на странице
- Мало текста, чёткая иерархия
- Блоки по смыслу "перелистываются"
- CTA каждые 1–2 экрана
- Приоритет: конверсия
- Ориентиры: sercons, il73 
- Делаем проще и быстрее

## 1. Общие параметры

| Параметр              | Значение                    |
| --------------------- | --------------------------- |
| Тип                   | Лендинг (1 экран ≈ 1 блок)  |
| Ширина контента       | 1200–1320 px                |
| Сетка                 | 12 колонок, gutter 20–24 px |
| Отступы между блоками | 80–120 px                   |
| Отступы внутри блоков | 24–40 px                    |

## 2. Стиль

- **Цвета:**
	- основной — синий (B2B, доверие)
	- акцент — более яркий синий или золотой (CTA)
	- фон — белый / светло-серый
- **Типографика:**
	- крупные плотные заголовки
	- читабельный текст
	- максимум 2 шрифта
- **UI:** 
	- крупные кнопки
	- радиус 8–12 px
	- минималистичные иконки (emoji в MVP)

## 3. Логика страницы

Каждый блок ведёт к действию → `Рассчитать`

### Блоки

1. **Hero** 
	- заголовок
	- подзаголовок
	- CTA "Рассчитать" (первична)
	- "Консультация" (вторична)
	- опционально иллюстрация/emoji.
2. **Квиз**
	- карточка 600–800 px
	- прогресс сверху
	- один вопрос на экран
	- крупные кнопки-ответы (1, 2 колонки).
3. **Сегментация**
	- 3 кликабельные карточки в ряд
4. **Преимущества**
	- 5 пунктов в ряд или сетка 2×3 (иконка + заголовок + 1–2 строки).
5. **Процесс**
	- 5 шагов (номер, короткий текст).
6. **Кейсы**
	- карточки: тип товара, срок, результат (результат выделить)
7. **Риски**
	- контрастный/серый блок
	- 4–5 пунктов
8. **Доверие**
	- логотипы
	- документы
	- лаборатории
	- сетка 3–4
9. **CTA**
	- центр: заголовок, подзаголовок, кнопка
10. **FAQ**
	- аккордеон
	- ширина 800–1000 px
11. **Финал** (мини-форма)
	- телефон
	- имя
	- кнопка "Оставить заявку"

## 4. Мобильная версия

- Все блоки в одну колонку
- Квиз по ощущению полноэкранный
- Основные кнопки на всю ширину
## 5. Критично
- Квиз выделен визуально
- Не перегружать текстом
- Считываемость блока за 3–5 секунд
